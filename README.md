# Fingerprint Attendance System

This project is a biometric attendance system that records attendance using a fingerprint sensor.

## Components Used
- Arduino Uno
- Fingerprint Sensor
- LCD Display
- Jumper Wires
- Power Supply

## Working
1. User places finger on the sensor
2. Fingerprint is matched with stored data
3. If matched, attendance is recorded

## Applications
- College attendance
- Office attendance
- Secure access systems
